Porque c#?
- Linguagem completa
- Maturidade (2001)
- Mercado
- Aplicabilidade (Games, mobile, web, desktop, console)
- Comunidade

Código Gerenciado
- Significa que sua execução depende de um gerenciador
- Também conhecido como Runtime
	- No caso do C# é CLR Runtime
- Gerencia memória, segurança entre outros recursos

CLR (é a sigla para Common Langage Runtime)

No ecossistema do .NET 
A compilação inicial gera um código intermediário
	- Intermediate Language (IL)

C# > IL > Código de maquina
JIT (Just in Time) - Processo de conversão do IL para binário é conhecido como JIT

C# > IL --> JIT > 01
